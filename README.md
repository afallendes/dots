# Dotfiles

This repository contains my configuration files. The `setup.sh` script
automates the process of creating symbolic links to these files in their
respective locations.

## Configurations

The following tools and applications are supported:

- **Bash**: `~/.bashrc`
- **Fish**: `~/.config/fish/config.fish`
- **Git**: `~/.gitconfig`
- **Micro**: `~/.config/micro/settings.json`
- **Starship**: `~/.config/starship.toml`
- **Vim**: `~/.vimrc`
- **Visual Studio Code**: `~/.config/Code/User/settings.json`
